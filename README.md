# The Pharmacy (AI Prototype)

A fictional retail pharmacy website prototype demonstrating AI chatbot integration for customer queries.

## Features
- Responsive retail-style frontend
- AI chatbot powered by GPT-4.1 mini
- Product recommendation system
- Purpose-based query handling (immunity, sleep, joints, etc.)
- Fully fictional dataset and branding

## Disclaimer
This is a simulation for demonstration purposes only.
No medical advice is provided.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: FastAPI
- AI: OpenAI API (GPT-4.1 mini)

## Deployment
Frontend: GitHub Pages  
Backend: Render
